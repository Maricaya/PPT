\tocSubSec{Instalação}

A instalação do \textbf{\glsShortUnder{sass}} normalmente é realizada através do \textbf{\glsShortUnder{npm}} ou \textbf{Yarn}, porém existem outras formas de realizar a Instalação, por exemplo em \textbf{macOS} é possível instalar recorrendo ao \textbf{HomeBrew}.

Ainda assim, com o \textbf{\glsShortUnder{npm}} ou \textbf{Yarn}, este pode ser instalado globalmente ou apenas instalado em cada projeto de forma individual. Os comandos que se seguem apresentam as duas formas de Instalação com \textbf{Yarn} e \textbf{\glsShortUnder{npm}}.

\begin{itemize}
	\item \textbf{Globalmente:}
	\begin{itemize}
		\item \textbf{Com NPM:} ~\verb|npm install -g sass|;
		\item \textbf{Com Yarn:} ~\verb|yarn global add sass|;
	\end{itemize}

	\item \textbf{Por Projeto:}
	\begin{itemize}
		\item \textbf{Com NPM:} ~\verb|npm install sass|;
		\item \textbf{Com Yarn:} ~\verb|yarn add sass|;
	\end{itemize}
\end{itemize}
